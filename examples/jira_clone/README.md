# Jira Clone

A simplified Jira clone built with the [Gea framework](https://github.com/dashersw/gea), demonstrating components, stores, routing, and the @geajs/ui component library.

This project is a port of [oldboyxx/jira_clone](https://github.com/oldboyxx/jira_clone) (originally written in React) to Gea.

## Running

```bash
npm install
npm run dev
```

## Stack

- **@geajs/core** — reactive components and stores
- **@geajs/ui** — pre-built UI primitives (Dialog, Button, Avatar, etc.)
- **Vite** — dev server and build tooling with `@geajs/vite-plugin`
- **Tailwind CSS** — utility-first styling
