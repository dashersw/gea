GEA Teleport Demo

This demo shows how the Teleport component can move content to different DOM locations while maintaining component state and reactivity.

Features Demonstrated:

  • Modal dialogs teleported to document body
  • Notifications teleported to a fixed position
  • Sidebar content teleported outside main layout
  • Conditional teleporting (enabled/disabled)
  • Reactive selector changes
  • Zero-reset: Components maintain state when teleported
